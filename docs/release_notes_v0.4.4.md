# Release Notes — v0.4.4

リリース日: 2025-11-27

## 重要な変更点

### Release 更新
- v0.4.3 での古いバイナリ参照を修正し、最新のインストーラファイルを正確に指すよう改善。
- CDN マニフェスト (`latest.yml`) の整合性を確保し、ダウンロードリンクが常に最新バージョンを提供するよう修正。

### バージョン管理
- ビルドシステムの最適化により、バージョン番号とアーティファクト命名の整合性を向上。

## バグ修正
- 古いバージョンのインストーラがキャッシュされていた問題を修正。
- CDN 配置時の マニフェスト更新を確実にするよう強化。

## 既知の注意点
- 既に v0.4.3 をインストールしている場合は、自動更新により v0.4.4 へ更新されます。

## 次回予定
- Remote exec の screenshot オプション出力の拡張。
